# sales-data-analysis
Sales data analysis project focused on identifying revenue drivers, customer behavior, and market trends using Excel.

## Overview
This project analyzes sales data to identify key revenue drivers, customer behavior, and market performance. The goal is to extract actionable insights that can support business decision-making.

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Data Visualization  

## Key Insights
- Revenue growth is primarily driven by increased order volume rather than pricing  
- Strong seasonality observed, with the second half generating ~40% higher revenue  
- Medium-sized orders dominate (~60% of revenue), making them the main performance driver  
- Revenue is moderately concentrated, with top 3 markets contributing ~63%  
- Customer base is well-diversified, with the top client contributing ~6% of total revenue  

## Assumption
Based on historical seasonality, stronger performance is expected in the second half of 2005. However, due to missing data, this should be treated as an assumption rather than a confirmed trend.

## Analysis Areas
- Revenue trends over time  
- Seasonality analysis  
- Market (geographic) performance  
- Customer concentration and retention  
- Order size (Small / Medium / Large) impact  

## Data Limitations
- 2005 contains only partial data (first 5 months)  
- Some conclusions are based on comparison with full-year data from 2003–2004
